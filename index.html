About Me

My name is Constantinos Iasonides. I love creating games, swimming, adventures in nature and action games like paintball. Currently I doing my bachelor's at Hanzehogeschool of Groningen in the Netherlands. My bachelor course is Game Design and Development.

In Media courses I learned how to create magazines, manage Photoshop, create pictures, change layers and colors. Also at the ICT courses, I learned how to use Microsoft Office and also learned how technology changed oaver the years.

Now, during the classes I learn how to use Illustrator, Photoshop, build websites, create and design logos. I learn how to use JavaScript and also create some games on my own.
